## Conversion Funnel

### The Path:
1. **See ad** → 2. **Click** → 3. **Land on homepage** → 4. **Scroll, read pricing** → 5. **Click "Get Started"** → 6. **Choose plan** → 7. **Pay** → 8. **Fill intake form** → 9. **You build** → 10. **They refer friends**

### Where People Drop Off (Fix These):
- **Homepage to Pricing:** Make pricing visible IMMEDIATELY (add pricing link in nav)
- **Pricing to Signup:** Add countdown timer "🔥 5 spots left this week"
- **Signup to Payment:** Make plans SUPER clear, no confusion
- **Payment to Intake Form:** Send email reminder if they don't fill it out within 24 hours

## Optimization Checklist (Weekly)

### Track These Metrics:
- **CTR (Click-Through Rate):** Aim for 1.5-3%
- **CPC (Cost Per Click):** Aim for $0.50-2.00
- **Conversion Rate:** Aim for 1-3%
- **CAC (Customer Acquisition Cost):** Aim for under $150

### If CTR is Low (<1%):
- Change creative (new images/video)
- Test new hook angles
- Improve ad copy

### If Conversion Rate is Low (<1%):
- Add more trust signals to landing page
- Simplify pricing
- Add live chat
- Test urgency ("5 spots left this week")

### If CAC is High (>$200):
- Pause underperforming campaigns
- Double down on winners
- Improve targeting (narrow audience)

## Red Flags to Avoid

❌ **Don't:**
- Run ads without testing checkout flow first
- Spend $100/day before proving $20/day works
- Target worldwide (stick to USA for now)
- Use generic stock photos (use real portfolio examples)
- Forget to set up conversion tracking (Facebook Pixel + Google Analytics)

✅ **Do:**
- Start small ($20-30/day), scale what works
- Test multiple ad angles
- Respond to every inquiry within 1 hour
- Overdeliver on first 10 customers (they're your marketing)
- Ask for referrals from happy customers
